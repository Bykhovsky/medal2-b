This fork modernizes Medal’s Luau decompiler for newer Roblox bytecode. It extends support from bytecode versions 4–6 through version 9, adds newer constants and opcodes, and improves error handling so malformed or host-wrapped bytecode returns a useful error instead of crashing the server worker.
